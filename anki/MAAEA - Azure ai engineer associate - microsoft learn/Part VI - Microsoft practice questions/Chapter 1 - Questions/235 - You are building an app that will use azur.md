========== Question ==========  

### You are building an app that will use Azure AI Vision to detect the presence of people in a video feed.

Which Azure AI Vision feature should you use?

Select only one answer.

-   face detection

-   Image Analysis

-   optical character recognition (OCR)

-   Spatial Analysis  

========== Answer ==========  

---

###### 1. Spatial Analysis

The only visual feature that provides this capability is Spatial Analysis, as OCR, Image Analysis, and face detection are not meant to analyze the presence of people in a video feed.
