========== Question ==========  

### You create an Azure AI Foundry Content Safety solution to monitor the text input of a chat forum.

You discover that the solution returns a severity level of 0 for various instances of harmful text.

Which type of accuracy error is returned by the solution?

Select only one answer.

-   a false negative

-   a false positive

-   a true negative

-   a true positive  

========== Answer ==========  

---

###### 1. a false negative

A false negative arises when the model fails to identify harmful content and returns a severity level of 0. There are four accuracy error types, and a false negative indicates incorrectly accepted content in the text.
